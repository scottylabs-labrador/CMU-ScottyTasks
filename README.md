# ScottyTasks

A gamified to do and habit tracking app. Earn coins by completing tasks and keeping up with habits. Coins allow you to upgrade your scottydog.

## Run on Expo Go

1. Install dependencies

   ```bash
   npm install
   ```

   ```bash
   npm install firebase
   ```

2. Start the app

   ```bash
   npx expo start
   ```
